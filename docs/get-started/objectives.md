# Objectives & Success Criteria

| Objective | Measurable criterion |
|---|---|
| Prove walletless UX | The demo completes with no wallet, no address, no seed phrase, no gas fee visible to the user |
| Prove the flow is real | A live webhook from the PSP sandbox triggers a real Hedera testnet transaction |
| Prove settlement | The stablecoin transfer is visible on HashScan during the presentation |
| Prove payout | The payout sandbox returns a success receipt for a local-currency credit |
| Prove verifiability | A Mirror Node query returns the compliance anchor without touching our API |
| Prove tamper-evidence | Altering the off-chain compliance record breaks hash verification |
| Prove privacy | No personal data, no KYC document, no name reaches the ledger — only a hash |
| State the limits | The report names every claim in this space that does not survive the regulation |
